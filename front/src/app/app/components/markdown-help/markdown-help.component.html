aide de mise en forme

/!\ La gestion du markdown est en cours de développement, il est possible que certaines fonctionnalités ne soient pas encore disponibles ou que des bugs soient présents.

Les notes utilisent le format Markdown pour mettre en forme les textes, similaire à Discord.
Si vous savez donc mettre en forme sur Discord, vous savez mettre en forme un texte sur le site.

Le format Markdown est une façon d’écrire du contenu pour le web. Voici comment procéder : il suffit d’ajouter simplement quelques caractères avant et après le texte souhaité pour modifier votre texte. Passons en revue quelques exemples.

Italique : *italique* ou _italique_
Italique souligné : __*italique souligné*__
Gras : **gras**
Gras souligné : __**gras souligné**__
Italique gras : ***italique gras***
Italique gras souligné : __***italique gras souligné***__
Souligné : __souligné__
Barré : ~~barré~~

Titre
Pour créer un titre, il vous suffit d'inclure un nombre spécifique de caractères dièse/dièse (#). Utilisez (#) pour un gros titre, (##) pour un titre plus petit ou (###) pour un titre encore plus petit comme premier(s) caractère(s) d'une nouvelle ligne pour créer un titre.
